# UPAS — READY TO USE

## الإصدار
**UPAS v1.0.0-RC1** — Release Candidate 1

## ما الذي يعمل

### المحرك الحسابي
- **تحليل الانفجارات** — ضغط الحادثة، الضغط المنعكس، الضغط الديناميكي، المدة، الدافع
- **صدمة الأرض** — الإزاحة، السرعة، التسارع، الإجهاد الأرضي
- **الاستجابة الإنشائية SDOF** — عزم الانحناء، قوة القص، الإزاحة القصوى، نسبة الانحراف
- **التصميم الإنشائي** — سماكة مطلوبة، تسليح مطلوب As، SF الانحناء والقص لكل عنصر
- **التحقق من الاختراق** — NDRC perforation + scabbing
- **التحقق من الانحراف** — δ/L لكل عنصر

### واجهة المستخدم (6 شاشات)
| الشاشة | الحالة | الوظيفة |
|--------|--------|---------|
| لوحة التحكم | ✅ | عرض المشاريع، تحميل مشروع تجريبي، مواد ومتفجرات وأنواع تربة |
| مشروع جديد | ✅ | إنشاء مشروع باسم وموقع ووصف → الانتقال المباشر إلى التحليل |
| التحليل | ✅ | عرض ثلاثي الأبعاد + نماذج المدخلات + نتائج + تقرير مهني |
| قاعدة البيانات | ✅ | متفجرات، مواد، أنواع تربة، منشآت، معايير، مشاريع |
| الإعدادات | ✅ | اللغة، السمة، الحفظ التلقائي، تفضيلات التقرير |
| حول البرنامج | ✅ | معلومات الإصدار، الوحدات الهندسية، التقنيات، المطور |

### التقرير المهني (Professional Report)
- غلاف المشروع مع حالة التصميم
- جدول المحتويات
- أساس التصميم (10 بنود مع المراجع)
- ملخص التهديد (TNT، مسافة، سرعة Detonation)
- معاملات الانفجار (6 قيم)
- نتائج التصميم لكل عنصر (سقف، جدران، أرضية)
- جدول ملخص التصميم مع SF ملون
- مصفوفة التحقق (4 فحوص × 3 عناصر)
- تفاصيل العنصر الحاكم مع النمط الحاكم (عربي)
- معرض أدوات التصدير (طباعة A4، تصدير ZIP)

### التصدير
- **طباعة A4** — تنسيق مهني جاهز للطباعة
- **حزمة التصدير ZIP** — تقرير + بيانات + معايير
